Para executar o projeto:
1 - Criar um banco de dados Java DB com os seguintes parametros:
Nome: LocadoraBD2
usuario: root
senha: root

2 - Copiar os jars dos projetos LivrariaJSF e LoginJSF

3 - Executar o arquivo que popula o banco de dados, encontrado em:
dsw-atividades/atividade02/src/java/br/ufscar/dc/dsw/dao/CriaUsuarios.java

4 - Executar o projeto no Netbeans
